The Problem

Every mining pool worldwide optimizes block templates for the same signal: current fee rate visible in the mempool. By the time a high-fee transaction is broadcast, the value is competed away — pool template builders converge on the same fee/vB threshold within milliseconds.

For DATUM-native operations, this convergence is even sharper. Miners constructing their own templates need an edge that isn't already priced into the shared mempool consensus.

The PreBlock Signal

PreBlock reads a second, slower-moving signal: UTXO dormancy age. When a Bitcoin that has been dormant for multiple years starts moving, the sender is either a large institutional player, a long-term holder, or somebody recovering lost keys — the transaction tends to carry higher value than fee/vB alone suggests.

Our engine surfaces these dormancy events 30 to 60 seconds before mempool consensus — using peer-observation of raw transaction propagation on a Bitcoin Knots full node.
